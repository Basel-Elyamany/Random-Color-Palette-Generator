# 🎨 Color Palette Generator

A simple JavaScript app that generates random colors, allows copying HEX codes, adding new colors, and deleting them.

Live URL: [Here](https://basel-elyamany.github.io/Random-Color-Palette-Generator/)

---

## 🚀 Features

- Generate random colors
- Copy HEX color code to clipboard
- Add new color boxes dynamically
- Delete any color box

---

## 🛠️ Technologies

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📜 Main Functions

### 🔹 generateRandomColors
- Generates random HEX colors
- Applies them to `.color` elements
- Updates `.hex-value` text
- Activates copy buttons

---

### 🔹 activateCopeBtn
- Adds click event to copy color code

---

### 🔹 activateDeleteBtn
- Adds click event to delete color box

---

### 🔹 addNewColor
- Creates a new color box
- Inserts it before the "Add Color" button
